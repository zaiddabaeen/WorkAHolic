# WorkAHolic

Ever wondered how much you are working a week?

Use the beautiful Lollipop-enabled WorkAHolic app to help you measure how much you work per week. 

It showcases Android Lollipop 5.0 features such as:
1) CardViews
2) Progress Bars
3) Floating Button
4) UI Colors
and others...
